Privacy Policy for Aura Bot

1. Information Collection

Aura collects certain user data to function effectively on Discord. This data includes:

  • Discord usernames and IDs
  • Date of joining a server where Aura is active

Aura does not log chat messages or any other interaction details beyond what is necessary to perform specific commands.

2. Data Usage

Data collected is used solely for enabling specific commands and enhancing the user experience on Discord. Aura does not use collected data for analytics, nor is it shared with third parties.

3. Data Storage and Retention

  • Storage: User data is stored in two ways:
    • In-memory data (temporary) during the session for real-time processing.
    • .json files (persistent) for data required over longer periods, allowing functionality across multiple sessions.
  • Retention: Collected data is retained as long as the bot is operational within a server. Data is deleted upon the bot’s removal or if explicitly requested by the user.

4. Data Security

While Aura stores data in .json files and temporarily in-memory, it currently does not encrypt data. Only bot administrators can delete stored data as part of maintenance procedures.

5. User Rights and Requests

  • Data Requests: Users can request access to, correction, or deletion of their data by joining our support server, "Aura Support," and notifying the bot’s administrative team.
  • Policy Updates: Any updates to this privacy policy will be communicated to users in the support server.

For more information or inquiries, feel free to join our support server.
